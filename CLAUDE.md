# Claude Code — empiecen acá

Este repositorio usa la metodología **KDD (Knowledge-Driven Development)**. Las reglas obligatorias
completas están en [.agents/AGENTS.md](.agents/AGENTS.md). También existe un puente equivalente en
[AGENTS.md](AGENTS.md).

Antes de escribir código: lean [DEFINITION.md](DEFINITION.md) (qué es TED y qué queda fuera de
alcance en esta primera versión), los contratos en `knowledge/contracts/`, y validen con
`python scripts/validate_contracts.py knowledge/contracts`. Este archivo es un puente, no duplica
contenido.
